# Pocket Patch

Pocket Patch is a lightweight wellness and habit app focused on small,
manageable actions called “patches.”

The goal is to help users make progress on hard days by offering
simple encouragement, reminders, and micro-tasks.

This project is being built as a learning experience while exploring
software development, GitHub workflows, and UI design.
## MVP Features
- Create a patch
- Mark patch as complete
- View active vs completed patches
- Persistent storage
